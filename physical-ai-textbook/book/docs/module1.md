# Module 1: ROS 2

ROS 2 is the standard software that connects all parts of a robot.

Key concepts:
- Nodes (programs that do one job)
- Topics (real-time messages)
- Services (request-reply)
- URDF (describes the robot body)

You will write Python code to control robots using rclpy.
